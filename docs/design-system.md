# Design System Foundation (Milestone 1)

## Visual Direction

- Industrial, modern, corporate
- Red accent with restrained use for CTAs and emphasis
- Steel/charcoal neutrals for trust and technical tone

## Color Tokens

- `brand`: `#C1252D`
- `brand-dark`: `#9F1A21`
- `steel-900`: `#111827`
- `steel-800`: `#1F2937`
- `steel-700`: `#374151`
- `steel-600`: `#4B5563`
- `steel-50`: `#F9FAFB`

## Typography

- Primary: `Public Sans`
- Fallback: `Segoe UI`, `Tahoma`, `Verdana`, `sans-serif`
- Clear heading hierarchy from `text-3xl` to `text-6xl` on hero

## Component Primitives

- Primary button
- Secondary button
- Section heading block
- Service card
- Feature card
- CTA banner
- Page hero

## Layout Standards

- Centered Tailwind `container`
- `section-space` utility for vertical rhythm
- `card-base` utility for consistent card shell
